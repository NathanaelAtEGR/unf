********
Glossary
********

.. glossary::

    CMake
        CMake is a compilation configuration platform that manages the build
        process in an operating system and in a compiler-independent manner.

        .. seealso:: https://cmake.org/overview/

    Tf Notification System
        Event Notification system included in the :term:`USD` library as part of
        the Tool Foundations (tf) module. It follows the `Observer pattern
        <https://en.wikipedia.org/wiki/Observer_pattern>`_.

        .. seealso:: https://graphics.pixar.com/usd/release/api/page_tf__notification.html

    USD
        Universal Scene Description (USD) is a framework for interchange of
        3D computer graphics data which focuses on collaboration and
        non-destructive editing.

        .. seealso:: https://graphics.pixar.com/usd/release/intro.html
